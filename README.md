# HR-ANALYTICS-DASHBOARD
📊 HR Analytics Dashboard
📌 Project Overview

The HR Analytics Dashboard is a data analysis project designed to help organizations understand employee trends, improve retention, and make better HR decisions.
This dashboard analyzes employee data such as attrition, job satisfaction, department performance, and salary distribution.

The goal of this project is to identify key factors affecting employee attrition and workforce performance using data visualization and analytics.

🎯 Objectives

Analyze employee attrition trends

Identify factors affecting employee turnover

Understand employee demographics and job roles

Visualize HR data for better decision-making
